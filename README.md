# Arquivo-Encriptador-de-Imagem
Programa feito para CUNHO EDUCATIVO! obs: Suas imagens não serão perdidas você escolhe a senha da encriptação

Como usar?
Primeiro extraia o arquivo .rar em uma pasta
Logo em seguida para se previnir crie uma pasta separada para o arquivo .exe
Coloque uma imagem qualquer (Pode ser mais de uma)
Abra o arquivo .exe e digite E para encriptar a imagem
Escolha a senha que quiser (Importante não se esquecer dela)
Pronto os arquivos foram encriptados

Como desencriptar?
Para desencriptar basta abrir novamente o programa .exe e deigitar a senha que você criou
Pronto os arquivos foram desencriptados
